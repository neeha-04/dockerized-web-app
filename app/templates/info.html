🎯 Project Overview

DevOps Monitor is a production-ready web application that gives you a real-time window into your system's health. It tracks live CPU usage, memory consumption, disk space, and network activity — all visualized through an interactive dashboard that updates every few seconds.

Behind the scenes, the app runs inside two Docker containers — one serving the Python Flask application, and another running Nginx as a reverse proxy — connected through a private Docker network. This mirrors exactly how real companies like Netflix, Uber and Airbnb deploy their applications in production environments, ensuring portability, scalability, and reliability across any server or cloud platform in the world.

🛠️ Technology Deep Dive

WHAT IS IT?

Docker is a containerization platform. Think of it like a lunchbox 🍱 — you pack your food (app) with everything it needs (libraries, settings) and it tastes exactly the same whether you open it at home, office, or cloud server.

WHY USE IT?

Without Docker, apps often break when moved between computers due to different Python versions, missing libraries, or OS differences. Docker eliminates this "works on my machine" problem forever.

HOW IT'S USED HERE

✅ Flask app runs in Container 1
✅ Nginx runs in Container 2
✅ Both talk via Docker network
✅ Docker Compose manages both

USED BY

Netflix, Uber, Spotify, Amazon, Google — every major tech company uses Docker in production.

WHAT IS IT?

Nginx (pronounced "Engine-X") is a reverse proxy and web server. Think of it like a receptionist 🏢 — all visitors come to the receptionist first, who then routes them to the right department (your Flask app).

WHY USE IT?

Flask's built-in server isn't meant for production — it can handle only one request at a time. Nginx handles thousands of simultaneous connections and passes them efficiently to Flask.

HOW IT'S USED HERE

✅ Listens on Port 80 (standard HTTP)
✅ Forwards to Flask on Port 5000
✅ Adds security headers
✅ Handles concurrent connections

USED BY

Powers 34% of all websites worldwide including Airbnb, GitHub, Instagram and Netflix.

WHAT IS IT?

Flask is a lightweight Python web framework. Think of it like a kitchen 🍳 — it gives you the tools to cook (build) your web app, but lets you decide exactly what to make and how.

WHY USE IT?

Flask is minimal, flexible and beginner-friendly. It doesn't force you into specific patterns, making it perfect for APIs, dashboards, and microservices.

HOW IT'S USED HERE

✅ Serves all HTML pages
✅ Provides REST API at /api/stats
✅ Blueprint architecture for clean code
✅ Jinja2 templates for dynamic HTML

USED BY

Pinterest, LinkedIn, Reddit and thousands of startups use Flask for their backends.

WHAT IS IT?

Gunicorn (Green Unicorn) is a WSGI server. Think of it like a manager 👔 — it creates multiple workers (employees) to handle many customer requests simultaneously so no one waits.

WHY USE IT?

Flask's development server handles ONE request at a time. Gunicorn spawns multiple worker processes, handling many requests in parallel — essential for production.

HOW IT'S USED HERE

✅ Runs 2 worker processes
✅ Sits between Nginx and Flask
✅ Handles concurrent requests
✅ Auto-restarts crashed workers

USED BY

Standard production server for Python web apps worldwide alongside uWSGI.

WHAT IS IT?

Python is a high-level programming language known for its clean, readable syntax. Think of it like English for computers 💬 — easy to write, easy to read, powerful to execute.

WHY USE IT?

Python has the largest ecosystem of libraries for web dev, AI/ML, automation, and DevOps. It's the #1 language for backend development and data science.

HOW IT'S USED HERE

✅ Core language for the entire app
✅ psutil library for system metrics
✅ datetime for timestamps
✅ platform for system info

USED BY

Google, NASA, Instagram, Spotify, Dropbox — Python is the world's most popular language.
